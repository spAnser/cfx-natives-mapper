# cfx-natives-mapper

Install modules with `yarn`

### Usage
```
USAGE
    node index.js [games]

ARGUMENTS
    gta5
    rdr3
```

You can then place the `[_docs]` into your resources folder and use a plugin like EmmyLua for VSCode to auto complete
